# NGO Foundation Website

## Overview

This project is a responsive NGO website developed as part of my **AI Web Development Internship Program**. The website was created for **InAmigos Foundation** with the goal of improving user engagement, accessibility, and overall user experience.

## Internship Task

This project was completed as an internship assignment focused on designing and developing a modern NGO website that effectively presents the foundation's mission, encourages volunteer participation, and simplifies the donation process.

## Features

* Responsive and modern design
* User-friendly navigation
* Donation section
* Volunteer information section
* Contact page
* Social media integration
* Clean and accessible layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* AI-assisted design and development tools

## Learning Outcomes

Through this project, I gained hands-on experience in:

* Frontend web development
* Responsive web design
* User Interface (UI) improvements
* User Experience (UX) considerations
* AI-assisted website development workflows
